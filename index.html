Midterm Revision App

This is a static HTML version of the Shiny app. All question generation, marking, and worked solutions run in each student’s browser.

Enter answers as fractions, decimals, or simple arithmetic, for example 2/3, 0.666666, or 0.05*0.99/(0.1445).

Hosting note: this file does not need Shiny Server or RStudio Connect. Upload it as index.html to any static web host.
